### 1.2 Кулинарное соревнование

ответ ≈ **26.83**

80 разных уровней. Вероятность выиграть для одного ((k - 1) / 79)^2. Тогда для всех это сумма ((k - 1) / 79)^2 где k от 1 до 80. формула суммы квадратов n(n + 1)(2n + 1) / 6 (80*159/6)/79 ≈ 26,83
