2.1
Сложность по времени: O(n), где n — длина строк, так как выполняется один проход по символам строк. Сложность по памяти: O(n), так как в худшем случае в словарях хранится до n различных соответствий. Оптимальность: Решение является оптимальным по времени, поскольку для проверки изоморфности необходимо как минимум один раз просмотреть все символы строк
